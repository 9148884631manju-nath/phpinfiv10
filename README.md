# phpinfi v10

**Opensource PHP Web Application Framework**  
by K. Manjunath, Code Designer

> A lightweight, JSON-driven PHP framework for building web apps without dependencies. Powers [relsusindia.com](http://relsusindia.com)

### **Features**

| Feature | Description |
| --- | --- |
| **JSON-Native** | Entire app configured via `lib/appconfig.json` |
| **Zero Dependencies** | Single file core. Works on PHP 8.1+ shared hosting |
| **Action Dispatcher** | Routes handled via JSON `action` → `switch` system |
| **Built-in SVG Engine** | Generate `<svg>` + CSS directly from JSON |
| **Theming System** | `webtheme` renders full pages from JSON layouts |
| **Asset Loader** | `webres` bundles fonts, CSS, JS from JSON manifests |
| **Session Helpers** | Custom session wrapper with `stemp` storage |
| **CSV/File Utils** | `writecsv`, `viewfiles`, `viewbyarray` built-in |

### **Requirements**
- PHP 8.1 or higher
- MySQL 5.0 or higher  
- Apache/Nginx
